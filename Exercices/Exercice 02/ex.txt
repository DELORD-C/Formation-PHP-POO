1. Créer une page html dans laquelle cous stockerez un formulaire de connexion (pseudo, password, submit)
2. Créer une base de donnée dans laquelle vous créez une table qui contient les champs suivants :
    - id (clé primaire et auto incrémentation : cocher AI)
    - pseudo
    - password
    - facebook
3. Créer un fichier index.php dans lequel vous incluez le fichier html précédemment créé
4. Créer une connexion à la base de donnée en php (objet PDO)
5. Démarrer la session
6. Faire en sorte que :
    Si l'utilisateur est connecté (variables session renseignées) on affiche son lien facebook et un lien de déconnexion
    Sinon
        Si La variable POST n'est pas vide : on vérifie la combinaison mot de passe / pseudo
            Si la combinaison est bonne, on connecte l'utilisateur (variables session) puis on rafraichi la page
            Sinon On rafraichi juste la page
        Sinon On affiche le formulaire
